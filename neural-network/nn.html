ReLU activation
ReLU(z) = max(0, z) — if the dot product is negative, the neuron outputs 0 (it doesn't fire). Positive values pass through unchanged. Apply ReLU to each neuron's dot product below.
Softmax output — prediction
The ReLU outputs feed into two output neurons — one for High UHI, one for Low UHI. Each computes a weighted sum of the ReLU values (logit), then softmax converts both logits into probabilities that sum to 1: P = eᶻ / (e^z_high + e^z_low)

How to read each layer
Dot product — each neuron multiplies every standardized input by its own weight and sums the results. Weights live on the edges, not inside the circles.
ReLU — max(0, z). Negative dot products are clipped to zero. This non-linearity lets the network learn patterns that a simple weighted sum cannot.
Softmax — converts two raw output scores (logits) into probabilities. The class with the higher probability is the network's prediction.
